# lowerCaseToUppperCase

This is a simple Java program that converts any user-entered text into UPPERCASE.

The program uses:
- Scanner for user input
- toUpperCase() method for conversion

## How it works
1. The user enters a sentence.
2. The program changes the sentence to uppercase.
3. The result is displayed on the screen.

## Example

Input:
hello yugesh

Output:
HELLO YUGESH

## Technologies Used

- Java
- Scanner class
- String methods
